bbt-cli，支持创建基于wepy，vue，react，原生小程序，普通H5，自定义模板的项目  

bbt-cli基于node.js v8.11.1，使用ES6，ES7的写法编写，摒弃了以往ES5旧的写法  

工作流程和vue-cli一样，从远端获取模板，根据用户的输入渲染模板  

## 使用

全局安装：  

```
  sudo npm i bbt-cli -g     

  bbt init my-project
```
